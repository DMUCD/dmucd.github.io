上网去冲浪

发表于 2022-06-04 更新于 2023-03-11 分类于 DMUCD 2.5k 2 分钟

男儿应是重危行，岂让儒冠误此生。

文档描述

  • 本文是作者上网冲浪时收藏的东西，可能需要耐心看
  • 为方便阅读，请添加网页目录生成 扩展 Smart TOC

温馨提示

  • 文中所列插件均能在Firefox扩展商店找到，未确认其他浏览器商店是否有
  • 文中所列脚本容易失效，可按关键词去脚本商店搜索新的

01. 搜索引擎

1.1 综合搜索

  1. Mirror List：发布谷歌搜索、谷歌学术、Github、维基百科等的镜像网站

  2. 访问Google搜索：

  • 马里共和国Google搜索官方站点：大陆用户可以直接访问
  • Google搜索镜像：镜像发布站、浙大镜像站（网站答案分别是“心灵之约、水朝夕、csxy@123”）
  • 添加 扩展 iGG谷歌访问助手后亦可访问
  1. Yandex：只有以图找图功能很好用
  2. F 搜：国产阉割版Goolge搜索
  3. 神马搜索：夸克浏览器在用，搜小说还可以

1.2 网盘搜索

专门搜索各大网盘资源，可能并不太好用……

1. 聚合搜索工具：

  • 千帆搜索、云盘资源网、易搜、橘子盘搜
  • 脚本 我是网盘管家婆
  • 插件 网盘全网搜索*1/*2

2. 阿里网盘搜索：

  • UP云搜、猫狸盘搜、云盘资源分享社区

3. 百度网盘搜索：

  • 比较好用：熊猫搜盘、乌鸦搜
  • 不太好使：兄弟盘、学霸盘、45256搜索

1.3 (TG)搜索

  • telegram中文搜索、TG喵、千帆FG搜索

1.4 其他搜索

  • iPlant 植物智：上传图片识别植物
  • The Pirate Bay：著名的海盗湾
  • 历史网页查询：Internet Archive、Archive Today

02. 插件脚本

添加脚本的步骤：

  1. 在扩展商店搜索 Tampermonkey 这个扩展程序并添加
  2. 在脚本商店搜索并添加想要的脚本即可

2.1 脚本商店

  • Greasy Fork：推荐使用
  • 油猴/暴力猴脚本插件中文网
  • 油猴脚本
  • Userscript.Zone

2.2 扩展商店

  1. 官方扩展商店：360极速、Firefox、Edge、Chrome

  2. 离线扩展下载：

  • Crx搜搜：提供Chrome,Edge,Firefox,Opera商店的插件
  • Crx4Chrome、极简插件、CrxDL、chrome666、扩展迷

03. 奶酪专区

浏览器深度开发技术达人

  • 奔跑中的奶酪：分享定制便携版Edge、Chrome、Firefox浏览器以及各种冲浪技巧
  • Awesome-Userscripts：奶酪-脚本推荐
  • Awesome-Extensions：奶酪-扩展推荐

04. GH 辅助

  1. 访问Github的辅助软件：UU加速器、Steam++、FastGithub、Steamcommunity 302

  2. GitHub加速代理下载站：

  • GHProxy、D-下载、GH-下载、GH-在线加速、GitHub directory downloader
  1. Github的辅助工具：
  • 脚本 Github 增强-高速下载
  • 脚本 GitHub 文件夹下载
  • 扩展 Gitako - GitHub file tree

05. 网页辅助

中国科学技术大学测速网站

5.1 视觉优化

  • 扩展 Auto Tab Discard：提升浏览器速度和减少内存占用
  • 扩展 Tree Style Tab - 树状标签页管理
  • 扩展 Smart TOC：在任意网页生成目录
  • 扩展 Clearly Reader：开启纯净无图阅读模式
  • 扩展 Dark Reader：适用于任何网站的黑暗主题
  • 扩展 广告拦截：AdGuard 广告拦截器、uBlock Origin

5.2 权限处理

  • 脚本 文本选中复制：解除复制限制
  • 脚本 网页限制解除(改)：可以解除禁止复制、剪切、选择文本、右键菜单的限制

5.3 视频播放

  • 脚本 HTML5视频播放器增强脚本：可在线倍速播放百度网盘视频
  • 扩展 Global Speed: 视频速度控制：可在线倍速播放百度网盘视频

5.4 图片查看

  • 扩展 浮图秀：快速查看页面中缩略图的“大图”，也可下载
  • 扩展 图片助手(ImageAssistant) 批量图片下载器

5.5 网页翻译

  • 扩展 网页翻译：Google/Yandex翻译、沙拉查词-聚合词典划词翻译

06. 下载视频

6.1 综合下载

  • m3u8格式下载：m3u8 DownLoad、m3u8 downloader
  • vget、online-downloader、Hitomi-Downloader
  • 脚本 无差别视频下载脚本

6.2 B 站下载

  • jijidown：傻瓜式B站视频下载器
  • B23Downloader：下载B站视频、直播、漫画
  • 哔哩下载姬downkyi
  • BilibiliVideoDownload：不支持付费视频和地区限制视频
  • 扩展 bilibili哔哩哔哩下载助手

07. 调教百度

  • 脚本 百度网盘一键重命名当前目录
  • 脚本 快捷键倍速，免VIP解锁倍速、画质
  • 度盘下载加速软件：爱奇艺万能联播、数码小站
  • 度盘在线解析下载：MCHZB-基于开源项目PanDownload网页复刻版
  • 度盘下载加速脚本：网盘直链下载助手
  • 加速下载微氪方法：去某多多或某宝①租个临时号 ②购买极速下载券
  • 使用百度云vip白嫖活动链接查找器可以每个月领1天或7天的VIP

08. 真互联网

部分素材来源：奔跑中的奶酪

1. 去网页重定向：

  • 脚本 自动跳转(重定向)到目标链接：Open the F**king URL Right Now、redirect外链跳转
  • 脚本 Anti-redirect：去除各搜索引擎/常用网站的重定向

2. 调教搜索引擎：

  • 脚本 百度谷歌必应链接缩短：缩短搜索引擎的冗长链接
  • 扩展 Buster：谷歌机器人验证码自动验证

3. 去除登陆弹窗：

  • 脚本 CSDN/知乎/哔哩哔哩/简书免登录去除弹窗广告
  • 脚本 骚扰拦截：自动拦截或删除下载弹窗、悬浮按钮等影响用户体验的元素
  • 脚本 拒绝二维码登录：默认使用账号密码登录，不出现二维码登录界面

4. 自动展开全文：

  • 脚本阅读全文/自动展开全文

09. 保存网站

  • HTTrack Website Copier：将整个网站离线到本地
  • 扩展 PDF Mage：将网页保存为PDF文件
  • 扩展 SingleFile：将一个完整的页面保存到单个HTML文件中
  • ArchiveWeb.page：该项目中提供的 扩展 可将网页保存为「.wacz」格式，用其提供的软件可以打开，也可以在线查看
