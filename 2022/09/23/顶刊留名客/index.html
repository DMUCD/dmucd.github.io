顶刊留名客

发表于 2022-09-23 更新于 2023-02-20 分类于 医疗废物 2.8k 3 分钟

新松恨不高千尺，恶竹应须斩万竿。

文档描述

  • 本文档中收集了一些编者认为对医学科研有帮助的信息
  • 为方便阅读，请添加网页目录生成扩展 Smart TOC

参与创作

  • 欢迎各位同行加入到本文的持续创作中来！
  1. 内容要求：任何你觉得对“搞科研/搞学术”有用的信息
  2. 内容投递：整理好后私发给阿岩即可
  3. 参与回馈：所有参与人员的署名将在文末特别鸣谢区列出
  4. 征集时限：永久有效

01. 白嫖文献

1.1 中国知网

1.1.1 白嫖数据库

  • 很多省市的大型图书馆都免费对外提供中文数据库：
  1. 如甘肃省图书馆免费对外提供万方数据库
  2. 在广西壮族图书馆注册账号，登陆即可使用其所购数据库
  3. 打开“支付宝”搜索“浙江图书馆”，可以注册读者证以使用其所购数据库
  4. 某省市的大型图书馆大厅可以办理用来访问其所购数据库的读者证

1.1.2 CAJ转成PDF

  • CAJ2PDF-qt：一款简单好用的格式转换软件
  • 通过中国知网海外版可直接下载PDF格式文献
  • 使用中国知网国内版时可配合脚本 知网硕博论文PDF下载

1.2 百度文库

  • 百度文库下载软件：文库下载器、xx文库下载器、稻壳阅读器、BdDownload、冰点文库
  • 百度文库下载脚本：百度文库下载、Wenku Doc DLer
  • 展开全文阅读脚本：百度文库净化
  • 如果你所在的机构购买了百度文库（高校版），那你就不用担心没有VIP了

1.3 文献传递

  1. 外文文献可前往SCI-hub搜索：镜像发布-官方、镜像发布-国内
  2. 在学术论坛进行文献求助：谷粉学术、科研通等
  3. 在医济群里礼貌求助总会有热心群众帮你快速找到的

1.4 找电子书

1.4.1 综合查询

  1. 易书主页 | 书籍检索：最好用的中文电子书搜索引擎
  2. ZLibrary-全世界电子资源最多的书库：
  • 官方：最新官网、官方APP
  • 民间：Zlib App Web API、Clibrary
  • 开源：Book Searcher、软件成品-PC、软件成品-安卓、网页成品1、网页成品2
  • 更多：Reddit-Zlibrary社区、奶大介绍-Github、书库备份-torrent
  1. Anna’s Archive：支持对Z-Library, Library Genesis, Sci-Hub进行跨库检索
  2. Library Genesis|帮助全人类知识无版权传播：镜像发布-国外、镜像发布-国内
  3. 鸠摩搜书 ：矮子里面选将军≈搜索国内电子书最好的引擎
  4. SSDOWN：全国图书馆参考咨询联盟/超星电子书库研究分享
  5. 三五块钱购买电子书：互助联盟、互助查询

1.4.2 标准指南

  • Standards Free Download：IEC、ANSI、IEEE、BS Standards
  • 搜索各种“指南”可使用微信搜索功能

02. 辅助工具

2.1 文献翻译

  • 文献翻译软件：知云文献翻译/Xtranslator、CopyTranslator、小绿鲸
  • 网页翻译插件：Google/Yandex翻译、沙拉查词-聚合词典划词翻译
  • 在线翻译网站：DeepL翻译、腾讯翻译君
  • CNKI学术翻译助手

2.2 文献管理

  • 两大主流文献管理软件：Endnote、Noteexpress
  • 手把手教你使用 EndNote X9/NoteExpress
  • 免费好用文献管理软件：Zotero

2.3 网页插件

  • easyScholar：显示期刊等级、在线翻译、期刊等级查询等
  • EasyPubMed ：专为新版PubMed设计的工具条，详细介绍见链接指向
  • SciHub CRX：显示IF、分区、引用次数、下载途径等，支持划词翻译

03. 检索导航

3.1 IF及分区查询

  • Journal Citation Reports
  • 中国科学院文献情报中心期刊分区表
  • 中文社会科学引文索引（CSSCI）
  • 第三方期刊智能查询系统：LetPub、MedSci

3.2 检索语言查询

  • 中图分类号查询
  • MeSH主题词查询:途径1、途径2

3.3 中文的数据库

  • 中国医院知识总库（CHKD）：支持MeSH主题词检索
  • 维普期刊：支持同义词扩展功能
  • 万方数据：推荐检索词（输入一段文字提取检索词）
  • 中华医学期刊全文数据库

3.4 英文的数据库

  • PubMed：期刊查询、临床查询、单批论文匹配
  • Web of Science：从文献之间相互引证关系的角度检索
  • Faculty Opinion：根据生物学和医学领域的全球领导者意见来评级和评价，并且由此推荐最好的科研文献
  • 谷歌学术搜索：镜像1、镜像2、镜像3、镜像发布

3.5 文献追踪助手

  • Stork：根据检索关键词定期推送相关的最新文献至邮箱中
  • LitSuggest：运用先进的机器学习筛选所文献，工作原理、使用教程

3.6 特色检索工具

  1. Connected Papers：获得与所检索文献相关的论文知识图谱，追踪文献的引用和被引关系

  2. Citespace：可视化文献分析软件（有人用它写了综述）

  3. PubMed第三方检索工具：`

  • 本地PubMed：界面与PubMed类似，支持标题翻译，一键申请全文(秒下载)

    注意：申请全文需要（注册）登录个人账号。若在校园网IP范围内，可以直接注册，若不在，需输入本单位的公共账号和密码（咨询本校图书馆或去网上搜），注册成功后使用不受IP限制

  • GM-根哥学术：支持按照IF（区间）、中科院分区、被引次数对检索结果排序；显示文献IF、分区

  • 银杏叶-PMOP：支持按照IF区间筛选检索结果；显示文献IF、分区(类型未知)

04. 文章写作

4.1 配图制作

  • 免费医学插图素材：SCI Draw、SMART、HealthIcons
  • 医学生物插图绘制：Biorender-❤、PPT插件“ScienceSlides”、科研绘-国内
  • 学术图表绘制工具：HIPLOT-网站、Graphpad Prism
  • 科技素材开放获取公益图库 （网站介绍）

4.2 查重降重

  • 笔杆网：连接校网的情况下可以每天免费查两次
  • 秘塔写作猫：对语句进行同义改写，降重可用，但请谨慎使用！

4.3 句子润色

  • phrasebank：学术写作语料库（该站付费PDF文件见医济网盘）
  • Grammarly：Free Online Writing Assistant
  • 微软爱写作-Aim Writing

4.4 文献复制

  • CopyPlusPlus：复制后自动合并多余换行，保留文献原有换行，自动合并空格（翻译鸡肋）

05. 文章发表

5.1 选刊网站

  • 英文期刊推荐：JournalFinder、JournalSuggester、JournalGuide

  • 中文期刊推荐：暂无

06. 其他查询

6.1 国自然基金

  • 官方 国家自然科学基金：共享服务网、2021申请代码、大数据知识管理服务门户（主要用于查询结题项目）
  • 非官方 国家自然科学基金查询网：可显示项目结题与否
  • LetPub网查询：可批量导出查询结果
