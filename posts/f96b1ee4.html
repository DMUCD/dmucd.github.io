账号注册成功后使用不受 IP 限制的医学科研平台：

  1. 本地PubMed：在官网基础上新增下载全文、显示期刊分区、跟踪检索式等功能 [详情]
  2. 迈特思创：集双语检索、全文获取、PDF全文翻译、期刊投稿指南等功能为一体 [详情]
  3. 聚合文献传递系统：输入 DOI / PMID / 题名出处索取文献，通过邮箱接收
  • 注册 请先咨询所在图书馆工作人员或找他校同学帮忙注册
  • 购买 V 阿岩 50元 代注册账号，可选方案有 1+3 or 2+3

02. 文献下载

2.1 知网万方

很多省市的大型图书馆都免费对外提供中文数据库：

  1. 如甘肃省图书馆免费对外提供「万方」数据库
  2. 在广西壮族图书馆注册账号，登陆即可使用其所购数据库
  3. 打开支付宝搜索“浙江图书馆”，可以注册读者证以使用其所购数据库
  4. 某些省市的大型图书馆大厅可以办理用来访问其所购数据库的读者证

脚本 免登录下载知网万方维普PDF/CAJ文件：

  1. 知网文章下载解析工具 [1]、知万维-学术下载解析 [2]
  2. CAJ转PDF工具：CAJ2PDF-qt、知网硕博论文PDF下载

2.2 百度文库

  • 软件 稻壳阅读器、xx文库下载器、BdDownload、冰点文库、文库下载器BY小叶
  • 脚本 百度文库下载、Wenku Doc DLer、百度文库净化·下载·解禁·VIP
  • 网站 文库下载助手：支持百度文库、道客巴巴、豆丁网
  • 如果你所在的机构购买了百度文库高校版，那你就不用担心没有 VIP 了

网站 Cuckooing：解析 CSDN、博客园、简书的文章，以 Markdown 格式下载到本地；同时支持百度文库提取、语雀知识库导出

2.3 文献传递

  1. 外文文献可前往 SCI-hub 搜索：镜像发布-官方、镜像发布-国内
  2. 在学术论坛进行文献求助：谷粉学术、科研通 等
  3. 在医济群里礼貌求助总会有热心群众帮你快速找到的

03. 找电子书

3.1 综合查询

  1. Anna’s Archive：支持跨库检索Sci-Hub、LibGen、Z-Library
  2. Library Genesis：帮助全人类知识无版权传播
  3. 鸠摩搜书 ：国内一众电子书网站最后的倔强
  4. 三五块钱购买电子书：互助联盟、互助查询

3.2 标准指南

  • 国家标准全文公开系统
  • Standards Free Download：IEC、ANSI、IEEE、BS Standards
  • 搜索各种「指南」可使用微信搜索功能

3.3 公开数据

中国卫生健康统计年鉴：可下载近10年的年鉴 PDF

04. 辅助工具

4.1 科研插件

  • 扩展 easyScholar、EasyPubMed、SangerboxNote
  • 脚本 SciHub CRX：显示 IF、分区、引用次数、下载途径等，支持划词翻译

4.2 文献翻译

  1. PDF 翻译软件：
  • 做任务才能用：Xtranslator🎯、小绿鲸
  • 完全免费使用：CopyTranslator、Pot、TTime、Bob
  1. 网页 翻译插件：
  • Translate Web Pages：使用 Google 或 Yandex 实时翻译网页
  • 网页双语对照翻译：Immersive-Translate、Kiss-Translator
  • 沙拉查词：聚合词典划词翻译
  1. 在线 翻译网站：DeepL翻译、腾讯翻译君、微信翻译
  2. CNKI学术翻译助手：用来翻译一些专业词汇很好使

4.3 文献复制

  • 脚本 文本选中复制：解除百度文库、中国知网等网站的复制限制
  • CopyPlusPlus：复制后自动合并多余换行，保留文献原有换行，自动合并空格
  • WebPlotDigitizer：从图像中提取数据，可用于散点图, 条形图，极坐标图, 地图等
  • Doc2X：上传图片/PDF 文件，OCR 识别后导出 MarkDown、LaTex、Docx 等格式文件
  • OCR 公式识别：Mathpix、Doc2X、SimpleTex、TexTeller、pix2tex

4.4 文献管理

广受好评的免费文献管理软件：Zotero

  • Endnote SUDA、WHU、NCU、USTB、JNU、JNMC、HENAU、CSU、CHD、NCEPU
  • NoteExpress NCEPU、SZU、GXU、CUPL、BJMU、ECNU、TJU、SWJTU、SCNU、SICAU、TongJi、BJTU、SUFE
  • 手把手教你使用 EndNote X9/NoteExpress

05. 检索导航

5.1 IF及分区查询

  • Journal Citation Reports
  • 中国科学院文献情报中心期刊分区表
  • 中文社会科学引文索引（CSSCI）
  • 第三方期刊智能查询系统：LetPub、MedSci

5.2 检索语言查询

  • 中图分类号查询
  • MeSH 主题词查询：途径1、途径2

5.3 中文的数据库

  • 中国医院知识总库（CHKD）：支持 MeSH 主题词检索
  • 维普期刊：支持「同义词扩展」功能
  • 万方数据：推荐检索词（输入一段文字提取检索词）
  • 中华医学期刊全文数据库

5.4 英文的数据库

  • PubMed：期刊查询、临床查询、单批论文匹配
  • Web of Science：从文献之间相互引证关系的角度检索
  • Faculty Opinion：根据生物学和医学领域的全球领导者意见来评级和评价，并且由此推荐最好的科研文献
  • 谷歌学术搜索镜像：[1]、[2]、[3]

5.5 文献追踪助手

  • Stork：根据「检索关键词」定期推送相关的最新文献至邮箱中
  • LitSuggest：运用先进的机器学习筛选所文献，工作原理、使用教程

5.6 特色检索工具

  1. Connected Papers：获得与所检索文献相关的论文知识图谱，追踪文献的引用和被引关系
  2. 文献计量学可视化分析工具：Citespace、VOSviewer
  3. PubMed第三方检索工具：GM-根哥学术、银杏叶-PMOP

06. 统计作图

Hiplot：一站式生物医学可视化分析平台

  • R 语言下载、镜像、R 语言包 ……（构建中）

  • JASP：基于R语言的统计分析软件，操作界面类似于SPSS

  • 数据科学中的R语言：四川师范大学研究生R语言公选课内容

  • SAS OnDemand for Academics：网页版免费 SAS 应用平台

07. 写作投稿

7.1 插图绘制

  • 免费医学插图素材：SCI Draw、SMART、HealthIcons、Bioicons、NICESCI
  • 医学生物插图绘制：Biorender🎯、MedPeer、ScienceSlides（PPT）、科研绘

7.2 查重降重

  • 学信网：应届毕业生享万方免费查重一次
  • 笔杆网：连接校网的情况下可以每天免费查两次
  • PaperYY：创建班级可得一天10次免费查重
  • 英文查重：iThenticate（30元/千字）、Sangerbox（Turnitin UK版 1.13/500字）
  • 秘塔写作猫：对语句进行同义改写，降重可用，但请谨慎使用！

7.3 句子润色

  • phrasebank：学术写作语料库（付费文档医济网盘已存）
  • Grammarly：Free Online Writing Assistant
  • 微软爱写作-Aim Writing

7.4 选刊网站

国际期刊预警名单-中国科学院文献情报中心

  • 英文期刊推荐：JournalFinder、JournalSuggester、JournalGuide

  • 中文期刊推荐：暂无
