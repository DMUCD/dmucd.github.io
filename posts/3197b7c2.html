不会选软件

发表于 2024-01-19 更新于 2024-01-20 分类于 DMUCD 6.4k 6 分钟

二八佳人体似酥，腰间仗剑斩愚夫

文档描述

  • 本文推荐了一些比较优秀的PC端软件，仅供参考
  • 在PC端打开该文档可显示目录，阅读效果更佳

温馨提示

  • 查找去广告版、破解版等请详阅博文「上传者之书」
  • 文中若无特别说明，戳超链接均会转跳至官方网站
  • 未添加超链接的，文中若无特别说明均为付费软件

01. 浏览器篇

以下是官方下载链接：

  • Firefox：公益组织 维护更新的，专注于隐私保护
  • 360极速：360系列良心之作，更适合国人习惯
  • Edge：chrome和IE双内核，能同步数据
  • Chrome：如果大陆地区能登陆那将是无敌的
  • 星愿：小众，设计新颖，支持Win、安卓、Mac三大平台数据同步

奶大定制的便携版本：

  • RunningCheese Chrome
  • RunningCheese Edge
  • RunningCheese Firefox

三大巨头其他下载页：

  • Firefox-Ftp：提供历史版本下载
  • Chrome-coolpage：提供canary、beta、dev版本下载
  • Edge-insider：提供canary、beta、dev版本下载

安卓端浏览器更新站：

  • Firefox-Updater：提供了 各种 浏览器的下载途径
  • Firefox安卓下载站：Ftp-Fenix、GH-Fenix、FirefoxCI

02. 下载工具

2.1 浏览器加速

从浏览器下载文件时可进行多线程下载，更快

  • neatdownloadmanager (NDM)：完全免费，不输IDM
  • Internet Download Manager (IDM)：同类型 Top1 [IDM Activation Script]

2.2 磁力下载器

  1. 专业级BT磁力下载：uTorrent、qBittorrent、BitComet(比特彗星)、Motrix
  • 有公网IP并配置BitTorrent Tracker速度才能上去
  • qBittorrent增强版、比特彗星全功能解锁豪华版[1.57] [1.98]

  1. 普通用户最常用的：迅雷、闪电下载、Free Download Manager(FDM)

  2. 新起之秀-PikPak：官网、官方APP、第三方客户端/网页端

  3. 付费神器：飞鸟下载器、电磁高手（在线播放）

03. 事项管理

  • Microsoft To Do：大厂出的、完全免费
  • 滴答清单：免费版本基本够用了
  • eDiary：完全 本地化 的电子日记本，支持加密

待办清单类软件的提醒功能真的鸡肋，还是钉钉和闹钟靠谱！

04. 系统增强

4.1 安全防护

  • 火绒安全：安静、会杀流氓弹窗、适合新手
  • 其他如 ESET、kaspersky更适合有重度需求的用户

4.2 效率办公

  • 多标签文件管理器：360文件夹、XYplorer、QTTabBar中文优化版
  • 开源 QuickLook：选中文件，单击 空格键 就能快速预览文件
  • Linkbar：添加多个任务栏
  • MouseInc：通过按住鼠标右键滑动鼠标执行一些快捷操作
  • 开源 高级剪贴板管理工具：Ditto、CopyQ
  • 开源 AltDrag：对任意窗口进行置顶、调整大小和透明度操作
  • 开源 BossKey：快速隐藏（开启）第三方程序界面 [1]

4.3 护眼软件

  • Iris Pro：专业防蓝光护眼软件
  • f.lux：根据时间自动调节色温亮度
  • Win10可开启系统自带夜间模式：设置/系统/屏幕/夜间模式

4.4 快速启动

分类管理程序快捷方式、文件（夹）、网页，桌面快速启动工具

  • 极速启动：支持贴边隐藏，音速启动的极简替代品
  • 类似Rolan的极简替代品：Lily、Lucy
  • 开源 GeekDesk：界面美观、支持自定义图标、背景

05. 讯息传递

  • TIM：QQ办公简洁版，可搭配微信/QQ/TIM防撤回&多开补丁
  • 网易邮箱大师：目前比较好的多邮箱账号管理软件
  • 邮箱推荐阿里邮箱：免费无广告、提醒及时、60GB容量
  • Proton Mail：提供私密、安全和加密的电子邮件账户

06. 文档编辑

6.1 Office及WPS

  1. Microsoft Office：推荐用Office Tool Plus安装，用HEU KMS Activator激活 [more]

  2. WPS Office：各大下载站有政府或机构定制版（专业版）、教育考试专用版

  3. 效率插件、工具：

  • OneKeyTools（PPT）、小恐龙（Word）、方方格子（Excel）
  • esDoc2Xls：将Word文档中表格信息导入至Excel [1]

6.2 仅 Markdown

  • Typora：非常优秀的轻量级Markdown编辑器和阅读器
  • 开源 Logseq：完全本地化的 大纲、双链 笔记
  • Zettlr：适合写作者和研究人员使用

6.3 文本代码编辑

  • 开源 Notepad++ 的替代品：Visual Studio Code、Notepad2、Notepad–、Notepad3

6.4 在线文档工具

使用在线文档为的是多平台同步访问编辑

  1. 常见工具：腾讯文档、金山文档、石墨

  2. 综合平台：新一代知识管理与协作平台，如 语雀（阿里）、飞书（字节）、FlowUs息流

  3. 大纲笔记：幕布，支持一键转思维导图，导出文档可使用 幕布批量导出助手

07. 同步共享

7.1 文档同步

将本地文档实时同步到云端

  • 坚果云：个人感觉是目前最好用的
  • Onedrive：微软的产品，同步速度有些慢
  • WPS-云文档：使用WPS Office的用户可以试试
  • 氪金大佬也可以试试各大主流网盘的「同步盘」功能

7.2 网盘共享

通过网盘分享文件更好管理

  • 蓝奏云盘：下载不限速、不限容量、下载无需登录、但上传文件不可超过100MB
  • 阿里云盘：下载不限速、送500GB+容量，第三方 版本有小白羊版v3、小白羊盘、小白云盘TV、蜗牛云盘、阿里云盘TV
  • 123云盘：下载不限速、下载无需登录（需要分享者提供流量）
  • 将网盘挂载到本地（变为本地硬盘）的软件：Alist、CloudNAS

临时储存/分享文件的网盘

  • Musetransfer、Ufile、File、Filecad、Tresorit Send、Userscloud、Zippyshare、Catbox

08. 导图绘制

8.1 思维导图

  • Xmind：国际知名,样式很多,但免费版导出有水印
  • MindManager：能与Microsoft office软件无缝集成
  • 幕布 和 语雀也可制作简洁（样式少）的思维导图

8.2 作流程图

  • 开源 Drawio：广受好评，Visio的替代产品
  • Microsoft Visio：不知为何，很多人想拥有
  • 网页 支持手绘示意图的工具：Excalidraw、Tldraw

09. PDF编辑

  • WPS Office个人版的PDF编辑功能超级好用
  • PDFelement：比Adobe Acrobat轻巧，功能却不输它
  • ABBYY FineReader：OCR功能强无敌
  • 其他如PDF-XChange、福昕PDF编辑器等
  • 批量生成PDF 书签（目录）：PdgCntEditor、QuickOutline

10. OCR识图

识别提取PDF或图片中的文字

  • ABBYY FineReader：主要用于制作扫描PDF文件的OCR版本
  • 开源 PandaOCR与Umi-OCR也都不错
  • 很多人推荐天若OCR，但个人觉得白描更好用，已支持正版🎯
  • QQ/TIM可使用快捷键 Ctrl+Alt+O 调用OCR识图：#截图功能提取版
  • 搜狗输入法也可添加 截屏-OCR扩展 调用OCR识图：#截图功能提取版

11. 文件管理

11.1 解压压缩

  1. Bandizip：简洁易上手，6.0 系列原生无广告|安装版/32位/64位便携版|
  2. 7zip：完全免费，极致压缩 No.1
  3. WinRAR：老牌压缩软件

11.2 文件处理

  • 文件批量重命名：菲菲更名宝贝、ReNamerPro
  • Lock Hunter：解除文件占用（移动到回收站）
  • Beyond Compare：文件 & 文件夹差异对比
  • 开源 ExifCleaner：清除图片、视频、PDF文件的元数据
  • Duplicate Cleaner Pro：专业的重复文件查找工具

11.3 文件搜索

比使用电脑自带的搜索功能快太多了

  • Everything：全盘极速搜索，可 批量 操作
  • Listary：连按两下 Ctrl 可快速拉起程序
  • 文件内容搜索软件：dnGrep、AnyTXT、Filelocator Pro

11.4 隐藏加密

  • 开源 VeraCrypt：磁盘文件加密工具
  • Wise Folder Hider Pro：支持文件 & 文件夹的隐藏与加密

11.5 文件传输

  • Fastcopy：复制包含大量小文件的文件夹时，速度很快
  • TeraCopy：文件复制工具，支持断点续传
  • 文件自动同步备份：FileGee、FreeFileSync、GoodSync
  • 开源 LocalSend：无需联网的跨平台文件互传工具

12. 修图剪辑

Topaz AI 系列软件可实现图片、视频的无损放大、锐化、降噪、抠图等

12.1 截图软件

  • QQ/TIM：使用快捷键 Ctrl+Alt+A 可调用截图，可截长图，也支持OCR、翻译功能
  • 微信：使用快捷键 Alt+A 也可调用截图，但只能实现简单截图功能
  • Faststone：付费软件，可以截长图
  • Snipaste：不能截长图，免费版基本够用
  • Screen to Gif：可以录制、编辑 GIF 格式动图

12.2 图像处理

  • CorelDRAW Graphics Suite(CDR)：矢量插画制作软件，比较有名
  • 开源 GIMP：被称为是Adobe Photoshop PS 的替代产品
  • 开源 Inkscape：专业的矢量图形编辑软件， Adobe Illustrator AI 的替代品
  • 开源 图像体积压缩软件：Imagine、caesium-image-compressor
  • Jpg-C：图像批量压缩、剪裁、调整大小、加水印工具
  • 图片批量漂白（校正）工具：PictureMagick、PM lite
  • 开源 图片无损放大工具、画质增强工具：Final2x、SuperImage

12.3 视频制作

  1. 视频录制软件：Bandicam、OBS Studio

  2. ZoomIt：用于技术演示的屏幕缩放、注释和录制工具🎯

  3. 视频无损切割/合并软件：LosslessCut、Avidemux

  4. 视频剪辑软件：

  • Adobe Premiere(Pr)的免费替代产品：DaVinci Resolve、Kdenlive
  • 新手可考虑 开源 的轻量级Shotcut 或 剪映
  1. Shanaencoder：全能视频编码 压制 软件（比小丸工具箱快）
  2. Moo0 视频压缩器：简单易上手（功能如其名）
  3. 开源 在屏幕上实时显示键盘操作😏：Carnac、Keyviz、KeyCastOW、NohBoard

12.4 音频处理

  • GoldWave：轻量级专业数字音频处理软件
  • 开源 人声、伴奏分离软件：Spleeter、Ultimate Vocal Remover
  • iZotope RX Audio Editor：可以识别并消除各种（特定）类型的背景噪音，同时支持人声、伴奏分离
  • Adobe Audition(AU)：可能时最全面的专业音频编辑工具
  • MP3TAG：可以编辑音频文件的标题，专辑，艺术家，歌词，封面等信息

12.5 字幕编辑

  • 开源 Subtitle Editor：视频字幕编辑软件
  • 文字转语音：淘金阁、Microsoft Azure、Balabolka、智影
  • 语音转文字：飞书妙记、音视频取文本工具（带时间轴）
  • Whisper：傻瓜式提取音频字幕工具
  • VAS视频加字幕：AI 一键视频加字幕、字幕翻译
  • Video subtitle extractor：将视频中硬字幕提取为外挂字幕文件(.srt格式)的工具

12.6 水印处理

  • DIKDIK Video Kit：批量去除、添加 滚动 水印； 追踪 人像添加马赛克等
  • Apowersoft Watermark Remover：添加&去除图片与视频的水印（水印支持文字、图片）

12.7 格式转换

  • 格式工厂：万能的多媒体格式转换工具箱，不输Wondershare Uniconverter
  • 开源 HandBrake： 免费的视频格式转换器

13. 影音播放

开源 EarTrumpet：独立控制不同的软件音量

13.1 视频播放

  • 两款优秀的国外的影音播放器：PotPalyer、VLC

  • 爱奇艺万能联播：国产的优质影音图播放器

13.2 音频播放

  1. Foobar2000：专业音频播放器
  • 建议普通用户搜索下载汉化版/美化版，推荐Asion汉化版
  • 开源 Foobox：Foobar2000 默认用户界面 (DUI) 的皮肤配置

  1. 千千静听：国产经典怀旧音乐播放器，国内软件站一般都有

  2. 开源 MusicPlayer2：支持在win10任务栏搜索框显示歌词

14. 看电子书

这里介绍的是看本地电子书的软件

  • Edge浏览器：自带PDF阅读器，方便快捷
  • 开源 Koodo-Reader：支持多达15种主流电子书格式， 内置笔记，高亮，翻译功能
  • Calibre：集电子书阅读器、格式转换器、编辑器于一身
  • 开源 Legado/阅读：安卓端最好的网络小说阅读器，支持自定义书源[1]、[2]、[3]、[4]

15. 电脑维护

15.1 软件卸载

专门的软件卸载工具能更好地清理程序残留文件

  • Geek Uninstaller：免费的轻量级软件卸载工具，媲美Total Uninstall
  • 开源 Bulk Crap Uninstaller：体积稍大但好用
  • 微软 Microsoft 支持和恢复助手：能够彻底卸载掉电脑上的Office应用
  • 微软 TroubleShooter：程序安装、卸载疑难问题解决工具
  • SoftCnKiller：高速下载器捆绑软件杀手（流氓软件检测清理）

15.2 垃圾清理

  • 火绒安全、电脑管家类软件都自带垃圾清理小工具
  • CCleaner：适合有一定电脑基础的人
  • 开源 Dism++：使用“空间回收”功能可以清理C盘空间
  • WizTree：号称“最快的磁盘空间分析工具”，能直观地展示所有文件和文件夹的大小
  • CleanMyWechat：自动删除 PC 端微信缓存数据，不会删除文字的聊天记录

该工具不会删除文字的聊天记录

15.3 系统维护

  • 开源 Ventoy：强大的启动U盘制作工具
  • 微PE工具箱：WinPE 装机、维护工具
  • 开源 Dism++：基于微软API开发的系统维护和优化工具
  • 傲梅分区助手：简单易用且免费的无损分区软件
  • 破解开机密码：NTPWEdit、Active@ Password Changer、Windows Login Unlocker
  • 微软常用运行库合集：包括了常用的 VB、VC++ 等，可以解决提示缺少 dll 文件等问题，使用修改版软件必备
  • CrystalDiskInfo：检测HDD/SSD硬盘信息（温度、通电次数、通电时间等）并评估健康状态

16. 玩机必备

  • 开源 TrafficMonitor：悬浮窗显示当前网速、CPU及内存利用率（支持任务栏显示）
  • 开源 PowerToys：微软开发的免费系统实用工具套件
  • Windows超级管理器：国产系统管理工具包，「隐私清理」模式更是C盘清理大杀器
  • 将系统还原到初始状态，保护系统不被更改：冰点还原、影子卫士
  • 开源 sandboxie：用其提供的沙盒虚拟环境安装运行可疑程序
  • 安装运行虚拟机：VirtualBox、VMware Workstation
  • Hasleo WinToUSB：把完整系统装进移动硬盘，让你的 Windows 随处可用 [便携版]

17. 外接设备

  • 开源 OpenBoard：面向教育工作者的跨平台交互式白板演示工具
  • 开源 Xournal++：跨平台手写笔记软件，支持数位板
  • KDE Connect：支持Linux、Android、Windows、macOS、iOS系统间设备互联
  • 开源 QtScrcpy：通过 USB/网络 连接，在PC端显示和控制Android设备（无需root权限）
  • 键鼠共享：Synergy可跨系统切换控制，Multiplicity 3是Windows系统间协同的最佳选择
  • Spacedesk：将iOS、Android设备变成Windows的第二屏幕
  • LittleBigMouse：使鼠标在不同分辨率的显示器之间平滑过度
