01. 浏览器篇

以下是官方下载链接：

  • Firefox：公益组织维护更新的，专注于隐私保护
  • 360极速：360 系列良心之作，更适合国人习惯
  • Edge：chrome 和 IE 双内核，能同步数据
  • Chrome：如果大陆地区能登陆那将是无敌的
  • 星愿：小众，设计新颖，支持 Win、安卓、Mac 三大平台数据同步
  • 基于 Chromium 开源内核的小众优质浏览器：百分、猫眼

奶大定制的便携版本：

  • RunningCheese Chrome
  • RunningCheese Edge
  • RunningCheese Firefox

三大巨头其他下载页：

  • Firefox-Ftp：提供历史版本下载
  • Chrome-coolpage：提供 canary、beta、dev 版本下载
  • Edge-insider：提供 canary、beta、dev 版本下载

安卓端浏览器更新站：

  • Firefox-Updater：提供了各种浏览器的下载途径
  • Firefox 安卓下载站：Ftp-Fenix、GH-Fenix、FirefoxCI

02. 下载工具

2.1 浏览器加速

从浏览器下载文件时可进行多线程下载，更快

  • neatdownloadmanager（NDM）：完全免费，不输IDM
  • Internet Download Manager（IDM）：同类型 Top1 [IDM Activation Script]

2.2 磁力下载器

  1. 专业级 BT 磁力下载：uTorrent、qBittorrent、BitComet（比特彗星）、Motrix
  • 有公网 IP 并配置BitTorrent Tracker速度才能上去
  • qBittorrent 增强版、比特彗星全功能解锁豪华版[1.57] [1.98]

  1. 普通用户最常用的：迅雷、闪电下载、Free Download Manager（FDM）

  2. PikPak：官网、官方APP、第三方客户端/网页端 [自动邀请程序]

  3. 付费神器：飞鸟下载器、电磁高手（在线播放）

03. 事项管理

  • Microsoft To Do：大厂出的、完全免费
  • 滴答清单：免费版本基本够用了 Premium
  • eDiary：完全本地化的电子日记本，支持加密
  • ManicTime：详细记录并分析电脑上每一款应用的运行轨迹

待办清单类软件的提醒功能鸡肋，还是钉钉和闹钟靠谱！

04. 系统增强

4.1 安全防护

  • 火绒安全：安静、会杀流氓弹窗、适合新手
  • 其他如 ESET、kaspersky 更适合有重度需求的用户

4.2 效率办公

  • 多标签文件管理器：360文件夹、XYplorer、QTTabBar中文优化版
  • Groupy：将软件窗口整合到一起，如同浏览器标签页般随意切换
  • QuickLook：选中文件，单击空格键就能快速预览文件
  • Linkbar：添加多个任务栏
  • MouseInc：通过按住鼠标右键滑动鼠标执行一些快捷操作
  • 高级剪贴板管理工具：Ditto、CopyQ
  • AltDrag：对任意窗口进行置顶、调整大小和透明度操作
  • BossKey：快速隐藏（开启）第三方程序界面 [1]

4.3 护眼软件

  • Iris Pro：专业防蓝光护眼软件
  • f.lux：根据时间自动调节色温亮度
  • Win10 可开启系统自带夜间模式 [设置/系统/屏幕/夜间模式]

4.4 快速启动

桌面快速启动工具：分类管理快捷方式、文件、网页

  • 极速启动：支持贴边隐藏，音速启动的极简替代品
  • 类似 Rolan 的极简替代品：Lily、Lucy
  • GeekDesk：界面美观、支持自定义图标、背景

05. 讯息传递

  • TIM 为 QQ 的办公简洁版，可搭配微信/QQ/TIM防撤回&多开补丁
  • 网易邮箱大师：目前比较好的多邮箱账号管理软件
  • 邮箱推荐阿里邮箱：免费无广告、提醒及时、60GB 容量
  • Proton Mail：提供私密、安全和加密的电子邮件账户

06. 文档编辑

6.1 Office及WPS

  1. Microsoft Office：推荐用 Office Tool Plus 安装，用 HEU KMS Activator 激活 [more]

  2. WPS Office：各大下载站有政府或机构定制版（专业版）、教育考试专用版

  3. 效率插件、工具：

  • PPT OneKeyTools、iSlide，Word 小恐龙、UUOffice，Excel 方方格子
  • esDoc2Xls：将 Word 文档中表格信息导入至 Excel [1]

6.2 仅 Markdown

  • Typora：非常优秀的轻量级 Markdown 编辑器和阅读器
  • Logseq：完全本地化的 大纲、双链 笔记
  • Zettlr：适合写作者和研究人员使用
  • 微信排版 Markdown 编辑器：Doocs、MDX Editor、Markdown Nice

6.3 文本代码编辑

  • Notepad++ 的替代品：Visual Studio Code、Notepad2、Notepad–、Notepad3

6.4 在线文档工具

使用在线文档为的是多平台同步访问编辑

  1. 常见工具：腾讯文档、金山文档、石墨

  2. 综合平台：新一代知识管理与协作平台，如 语雀（阿里）、飞书（字节）、FlowUs息流

  3. 大纲笔记：幕布，支持一键转思维导图，导出文档可使用幕布批量导出助手

07. 网盘共享

比较好用的百度网盘替代品

  • 蓝奏云盘：下载不限速、不限容量、下载无需登录、但上传文件不可超过 100MB
  • 阿里云盘第三方版本有小白羊盘、小白云盘TV、蜗牛云盘、阿里云盘TV
  • 123云盘：下载不限速、下载无需登录（需要分享者提供流量）
  • 将网盘挂载到本地（变为本地硬盘）的软件：Alist、CloudNAS

临时储存/分享文件的网盘

  • Musetransfer、Ufile、File、Filecad、Tresorit Send、Userscloud、Zippyshare、Catbox

08. 导图绘制

8.1 思维导图

  • Xmind：国际知名,样式很多,但免费版导出有水印
  • MindManager：能与 Microsoft office 软件无缝集成
  • 幕布和语雀也可制作简洁（样式少）的思维导图

8.2 作流程图

  • Drawio：广受好评，Visio 的替代产品
  • Microsoft Visio：不知为何，很多人想拥有
  • 支持手绘示意图的网站：Excalidraw、Tldraw、Handraw

09. PDF编辑

  • WPS Office 个人版的 PDF 编辑功能超级好用
  • PDFelement：比 Adobe Acrobat 轻巧，功能却不输它
  • ABBYY FineReader：OCR 功能强无敌
  • 其他如 PDF-XChange、福昕PDF编辑器等
  • 批量生成 PDF 书签（目录）：PdgCntEditor、QuickOutline

10. 截图OCR

  • 专业截图软件：Faststone 可以截长图，Snipaste 有个人免费版
  • 专业 OCR 软件：白描🎯、Umi-OCR、天若OCR、ABBYY FineReader、PandaOCR
  • 搜狗输入法和 QQ/TIM 截图功能提取版支持截（长）图、OCR
  • （截图）翻译 & OCR 集成软件：eSearch、Sunny、Bob

11. 文件管理

11.1 解压压缩

  1. Bandizip：简洁易上手，6.0 系列原生无广告安装版/32位/64位便携版
  2. 7zip：完全免费，极致压缩 No.1
  3. WinRAR：老牌压缩软件

11.2 文件处理

  • 文件批量重命名：菲菲更名宝贝、Flut Renamer
  • Lock Hunter：解除文件占用（移动到回收站）
  • Beyond Compare：文件 & 文件夹差异对比
  • ExifCleaner：清除图片、视频、PDF 文件的元数据
  • Duplicate Cleaner Pro：专业的重复文件查找工具

11.3 文件搜索

检索速度秒杀电脑自带的搜索功能

  • Everything：全盘极速搜索，可批量操作
  • Listary：连按两下 Ctrl 可快速拉起程序
  • 文件内容搜索软件：dnGrep、AnyTXT、Filelocator Pro

11.4 隐藏加密

  • VeraCrypt：磁盘文件加密工具
  • Wise Folder Hider Pro：支持文件 & 文件夹的隐藏与加密

11.5 文件传输

  • 文件复制工具：Fastcopy 复制大量小文件时速度优异，TeraCopy 支持断点续传
  • 文件同步备份：FileGee、开源 FreeFileSync、GoodSync
  • Syncthing [GUI]： 基于P2P 技术的文件远程自动同步工具
  • 同步备份云盘：坚果云、Onedrive、Dropbox，其他主流网盘
  • LocalSend：无需上传云端，跨平台局域网文件互传工具

12. 修图剪辑

支持 PSD、WebP 等多种格式的图片查看器：Honeyview、ImageGlass

12.1 图像处理

  • Screen to Gif：可以录制、编辑 GIF 格式动图
  • CorelDRAW Graphics Suite（CDR）：矢量插画制作软件，比较有名
  • 开源 GIMP：被称为是Adobe Photoshop（Ps）的替代产品
  • 开源 Inkscape：专业的矢量图形编辑软件， Adobe Illustrator（Ai）的替代品
  • 图像体积压缩软件：Imagine、Caesium、极速图片压缩器
  • Jpg-C：图像批量压缩、剪裁、调整大小、加水印工具
  • 图片批量漂白（校正）工具：PictureMagick、PM lite
  • 图片无损放大工具、画质增强工具：Upscayl、Final2x、SuperImage、Real-ESRGAN

12.2 视频制作

  1. 视频录制软件：ZD Soft Screen Recorder、Bandicam、OBS Studio

  2. ZoomIt：用于技术演示的屏幕缩放、注释和录制工具🎯

  3. 视频无损切割/合并软件：LosslessCut、Avidemux

  4. 视频剪辑软件：

  • Adobe Premiere（Pr）的免费替代产品：DaVinci Resolve、Kdenlive
  • 新手可考虑 开源 的轻量级 Shotcut 或 剪映[Pro]
  • FunClip：准确识别视频中的语音，基于识别结果的文字选段随心裁剪视频片段
  1. Shanaencoder：全能视频编码压制软件（比小丸工具箱快）
  2. Moo0 视频压缩器：简单易上手（功能如其名）
  3. 在屏幕上实时显示键盘操作😏：Carnac、Keyviz、KeyCastOW、NohBoard

12.3 音频处理

  • Audacity：专业数字音频编辑器 GoldWave 的替代品
  • 人声、伴奏分离软件：Spleeter、Ultimate Vocal Remover [在线]、Demucs
  • iZotope RX Audio Editor：可以识别并消除各种（特定）类型的背景噪音，同时支持人声、伴奏分离
  • Adobe Audition（Au）：可能时最全面的专业音频编辑工具
  • MP3TAG：可以编辑音频文件的标题，专辑，艺术家，歌词，封面等信息

12.4 字幕编辑

  • Subtitle Editor：视频字幕编辑软件
  • VAS视频加字幕：AI 一键视频加字幕、字幕翻译
  • Video subtitle extractor：将视频中硬字幕提取为外挂字幕文件 .srt 的工具
  • 视频翻译和配音工具：AI 视频翻译、配音、字幕生成等一站式处理
  • Video subtitle master：批量为视频生成字幕，并可将字幕翻译成其它语言

12.5 水印处理

  • DIKDIK Video Kit：批量去除、添加 滚动 水印；追踪 人像添加马赛克等
  • Apowersoft Watermark Remover：添加&去除图片与视频的水印（水印支持文字、图片）

12.6 格式转换

  • 格式工厂：万能的多媒体格式转换工具箱，不输 Wondershare Uniconverter
  • 开源 HandBrake： 免费的视频格式转换器

13. 影音播放

EarTrumpet：独立控制不同的软件音量

13.1 视频播放

  • 两款优秀的国外的影音播放器：PotPalyer、VLC

  • 爱奇艺万能联播：国产的优质影音图播放器

13.2 音频播放

  1. Foobar2000：专业音频播放器
  • 建议普通用户搜索下载汉化版/美化版，推荐 Asion汉化版
  • Foobox：Foobar2000 默认用户界面（DUI）的皮肤配置

  1. 千千静听：国产经典怀旧音乐播放器，国内软件站一般都有

  2. MusicPlayer2：支持在 win10 任务栏搜索框显示歌词

14. 看电子书

这里介绍的是看本地电子书的软件

  • Edge浏览器：自带 PDF 阅读器，方便快捷
  • 开源 Koodo-Reader：支持多达 15 种主流电子书格式， 内置笔记，高亮，翻译功能
  • Calibre：集电子书阅读器、格式转换器、编辑器于一身
  • Legado/阅读：安卓端最好的网络小说阅读器，支持自定义书源 [1]、[2]、[3]、[4]

15. 电脑维护

15.1 软件卸载

专业的卸载工具能彻底清理程序残留文件

  • Geek Uninstaller：免费的轻量级软件卸载工具，媲美 Total Uninstall
  • 开源 Bulk Crap Uninstaller：体积稍大但好用
  • 微软 Microsoft 支持和恢复助手：能够彻底卸载掉电脑上的 Office 应用
  • 微软 TroubleShooter：程序安装、卸载疑难问题解决工具
  • SoftCnKiller：高速下载器捆绑软件杀手（流氓软件检测清理）

15.2 垃圾清理

  • 火绒安全、电脑管家类软件都自带垃圾清理小工具
  • CCleaner：适合有一定电脑基础的人
  • Dism++：使用“空间回收”功能可以清理 C 盘空间
  • WizTree：号称“最快的磁盘空间分析工具”，能直观地展示所有文件和文件夹的大小
  • CleanMyWechat：自动删除 PC 端微信缓存数据，不会删除文字的聊天记录

15.3 系统维护

  • 开源 Ventoy：强大的启动 U 盘制作工具
  • 微PE工具箱：WinPE 装机、维护工具
  • Dism++：基于微软 API 开发的系统维护和优化工具
  • 傲梅分区助手：简单易用且免费的无损分区软件
  • 破解开机密码：NTPWEdit、Active@ Password Changer、Windows Login Unlocker
  • 微软常用运行库合集：包括了常用的 VB、VC++ 等，可以解决提示缺少 dll 文件等问题，使用修改版软件必备
  • CrystalDiskInfo：检测 HDD/SSD 硬盘信息（温度、通电次数、通电时间等）并评估健康状态
  • 360 驱动大师：专业解决驱动安装更新问题，网卡版适用无网络情况

16. 玩机必备

  • TrafficMonitor：悬浮窗显示当前网速、CPU 及内存利用率（支持任务栏显示）
  • PowerToys：微软开发的免费系统实用工具套件
  • Windows超级管理器：国产系统管理工具包，「隐私清理」模式更是 C 盘清理大杀器
  • 将系统还原到初始状态，保护系统不被更改：冰点还原、影子卫士
  • 开源 sandboxie：用其提供的沙盒虚拟环境安装运行可疑程序
  • 安装运行虚拟机：VirtualBox、VMware Workstation Pro
  • Hasleo WinToUSB：把完整系统装进移动硬盘，让你的 Windows 随处可用 [便携版]

17. 外接设备

  • 开源 OpenBoard：面向教育工作者的跨平台交互式白板演示工具
  • Xournal++：跨平台手写笔记软件，支持数位板
  • KDE Connect：支持 Linux、Android、Windows、macOS、iOS 系统间设备互联
  • QtScrcpy：通过 USB/网络连接，在 PC 端显示和控制 Android 设备（无需 root 权限）
  • 键鼠共享：Synergy 可跨系统切换控制，Multiplicity 3 是 Windows 系统间协同的最佳选择
  • Spacedesk：将 iOS、Android 设备变成 Windows 的第二屏幕
  • LittleBigMouse：使鼠标在不同分辨率的显示器之间平滑过度
