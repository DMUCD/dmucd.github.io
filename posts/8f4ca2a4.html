上网去冲浪

发表于 2024-01-19 更新于 2024-02-08 分类于 DMUCD 3.2k 3 分钟

男儿应是重危行，岂让儒冠误此生

文档描述

  • 本文是作者上网冲浪时收藏的东西，可能需要耐心看
  • 建议在PC端打开博客文章，可显示目录(TOC)

温馨提示

  • 文中所列插件均能在Firefox（火狐）浏览器扩展商店找到
  • 文中所列脚本容易失效，可按关键词去脚本商店搜索新的

01. 搜索引擎

破解软件专用搜索引擎「破解软件资源猎手 V3.0」

1.1 综合搜索

  1. Mirror List：发布谷歌搜索、谷歌学术、Github、维基百科等的镜像网站
  2. 网页代理：提供谷歌搜索、维基百科、Github等网站的代理服务
  3. Google搜索-浙大镜像站（网站答案分别是“心灵之约、水朝夕、csxy@123”）
  4. F 搜：国产阉割版Goolge搜索
  5. 神马搜索：夸克浏览器在用，搜小说还可以
  6. DuckDuckGo：国外知名匿踪搜索引擎

插件 ContextSearch web-ext：选中网页内容右键搜索

1.2 网盘搜索

专门搜索各大网盘资源，可能并不太好用……

  1. 聚合搜索工具：
  • 网站 千帆搜索、云盘资源网、易搜、橘子盘搜、PanSearch、数据搜索、迅搜搜
  • 脚本 我是网盘管家婆
  • 插件 网盘全网搜索[1]、[2]

  1. 阿里网盘搜索：玩偶哥哥、小纸条、UP云搜、猫狸盘搜、云盘资源、More

  2. 百度网盘搜索：熊猫搜盘、兄弟盘、学霸盘、45256搜索、西瓜搜搜

  3. 夸克网盘搜索：云盘4K、柚子追剧、趣盘搜

  4. 迅雷云盘搜索：YaPan

1.3 (TG)搜索

  • telegram中文搜索、TG喵、千帆TG搜索、TGSO

1.4 图片搜索

  • iPlant 植物智：上传图片识别植物
  • 聚合以图搜图工具：Reverse image Search [1] [2]（支持Google、Bing、Yandex、Baidu）
  • Lexica：搜索 AI 图库的类似图片
  • Immerse：支持通过绘制草图、文字描述搜索相似图片

1.5 其他搜索

  1. The Pirate Bay：著名的盗版资源检索网站
  2. 2tool.top：查询各大搜索引擎的网页快照
  3. WebArchive：定期收录并永久保存全球网站上可以抓取的信息
  4. ArchiveToday：手动保存网页、网页存档查询
  5. 给程序员准备的代码搜索引擎：开发者搜索、Cursor、Devv、Socode.Pro、Goobe、SearchCode、aiXcoder、grep.app

02. 插件脚本

添加脚本的步骤：

  1. 在扩展商店搜索 Tampermonkey 这个扩展程序并添加
  2. 在脚本商店搜索并添加想要的脚本即可

2.1 脚本商店

  • Greasy Fork🎯、油猴/暴力猴脚本插件中文网、油猴脚本、Userscript.Zone

2.2 扩展商店

  1. 官方扩展商店：360极速、Firefox、Edge、Chrome

  2. 离线扩展下载：

  • Crx搜搜：提供Chrome,Edge,Firefox,Opera商店的插件
  • Crx4Chrome、极简插件、CrxDL、chrome666、扩展迷

2.3 精选推荐

  • Awesome-Userscripts：实用 脚本 列表
  • Awesome-Extensions：实用 扩展 列表

03. GH 辅助

  1. 访问Github的辅助软件：
  • 网易UU加速器、Watt Toolkit、FastGithub、Steamcommunity 302
  1. GitHub加速代理下载站：
  • GitHub Proxy、D-下载、GH-下载、GH-在线加速、GitHub directory downloader
  1. Github的辅助增强工具：
  • 脚本 Github 增强-高速下载、GitHub 文件夹下载
  • 扩展 Gitako - GitHub file tree
  1. 修改本地hosts文件方式：
  • GitHub520、fetch-github-hosts、GitHub Hosts

04. 网页辅助

测速网站：东北大学、中国科学技术大学

4.1 视觉优化

  • 扩展 Auto Tab Discard：提升浏览器速度和减少内存占用
  • 扩展 Tree Style Tab - 树状标签页管理
  • 扩展 Smart TOC：在任意网页生成目录
  • 扩展 Clearly Reader：开启纯净无图阅读模式
  • 扩展 Dark Reader：适用于任何网站的黑暗主题
  • 扩展 广告拦截：AdGuard 广告拦截器、uBlock Origin
  • 脚本 自动无缝翻页：无缝加载下一页内容至网页底部（类似瀑布流）

4.2 权限处理

  • 脚本 文本选中复制：解除复制限制
  • 脚本 网页限制解除(改)：可以解除禁止复制、剪切、选择文本、右键菜单的限制
  • 书签 超级书签|油小猴：解锁网页右键/复制限制，显示星号密码，网页自由编辑，搜索扩展等

4.3 图片查看

  • 扩展 浮图秀：快速查看页面中缩略图的“大图”，也可下载
  • 扩展 图片助手(ImageAssistant) 批量图片下载器
  • 脚本 Search By Image：反向搜索网页图片，支持Google、Bing、Yandex、Baidu等众多搜索引擎

4.4 网页翻译

  • 扩展 Translate Web Pages：使用 Google 或 Yandex 实时翻译网页
  • 扩展 Immersive-Translate：沉浸式双语网页翻译
  • 扩展 沙拉查词：聚合词典划词翻译

05. 下载视频

5.1 综合下载

  • 主流媒体视频下载： vget、online-downloader、Hitomi-Downloader
  • m3u8 格式下载：m3u8 DownLoad、m3u8 downloader
  • 脚本 无差别视频下载脚本
  • 闪豆视频下载器：可下载国内各大视频平台的资源

5.2 B 站下载

  • 软件 jijidown、B23Downloader、Downkyi、BilibiliVideoDownload
  • 扩展 bilibili哔哩哔哩下载助手

06. 调教网盘

6.1 百度网盘

  • 脚本 百度网盘一键重命名当前目录
  • 脚本 快捷键倍速，免VIP解锁倍速、画质
  • 度盘在线解析下载：容易失效，多基于开源项目PanDownload网页复刻版
  • 脚本 度盘下载加速：网盘直链下载助手、KubeDown
  • 加速下载微氪方法🎯：去某多多或某宝①租个临时号 ②购买极速下载券
  • 使用百度云vip白嫖活动链接查找器可以每个月领1天或7天的VIP
  • 脚本 百度网盘秒传链接提取、百度网盘秒传转存助手、秒传解析网站

6.2 其他网盘

  • 脚本 （改）网盘直链下载助手：在网页端获取网盘文件真实下载地址，调用IDM等工具下载。支持百度/阿里/天翼/迅雷/夸克/移动六大网盘

6.3 辅助工具

  • 倍速播放网盘视频： 脚本 BD网盘视频播放器、 脚本 HTML5视频播放器增强、扩展 Global Speed

07. 调教腾讯

  • 扩展 QQ空间导出助手：用于备份QQ空间的说说、日志、私密日记、相册（原图）、视频、留言板、QQ好友、收藏夹、分享、最近访客为文件
  • 待测试：QQ空间相册下载神器、QZAlbumTool、QQ-Zone

08. 真互联网

部分素材来源：奔跑中的奶酪

  1. 去网页重定向：
  • 脚本 自动跳转(重定向)到目标链接：Open the F**king URL Right Now、redirect外链跳转
  • 脚本 Anti-redirect：去除各搜索引擎/常用网站的重定向
  1. 调教搜索引擎：
  • 脚本 百度谷歌必应链接缩短：缩短搜索引擎的冗长链接
  • 扩展 Buster：谷歌机器人验证码自动验证
  1. 去除登陆弹窗：
  • 脚本 CSDN/知乎/哔哩哔哩/简书免登录去除弹窗广告
  • 脚本 骚扰拦截：自动拦截或删除下载弹窗、悬浮按钮等影响用户体验的元素
  • 脚本 拒绝二维码登录：默认使用账号密码登录，不出现二维码登录界面
  1. 自动展开全文：
  • 脚本 阅读全文/自动展开全文

09. 保存网站

  • 将整个网站离线到本地：HTTrack Website Copier、Offline Explorer、仿站小工具
  • 扩展 PDF Mage：将网页保存为 PDF 文件
  • 扩展 SingleFile：将一个完整的页面保存到单个 HTML 文件中
  • ArchiveWeb.page：该项目中提供的 扩展 可将网页保存为 wacz 格式，用其提供的软件可以打开，也可以在线查看
